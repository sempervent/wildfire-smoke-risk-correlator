# Plume and dispersion

Smoke exposure in this repo combines **simple corridor plume heuristics** (`wind_v1`, `wind_grid_v2`) with an optional **Gaussian-style dispersion proxy** (`gaussian_v0`). None of these are regulatory dispersion models or public-health advisories.
