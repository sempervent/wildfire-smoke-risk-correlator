# Release v0.1.0 (candidate)

**v0.1.0** marks the first tagged engineering milestone for this repository: a **local-first Kafka + Spark + PostGIS** correlator with **smoke-risk scoring**, optional **dispersion / calibration scaffolding**, and **operator-facing SQL + Grafana** surfaces.

## What this release includes

- **Ingestion and normalization** of FIRMS hotspots, OpenAQ-style measurements, wind, and optional gridded weather into PostGIS-linked census geographies.
- **Risk models v1–v5** (configuration-dependent), **plume** heuristics (`wind_v1`, `wind_grid_v2`), and optional **Gaussian-style dispersion proxy** (`gaussian_v0`) — all explicitly **non-regulatory** engineering correlations.
- **Calibration / evaluation hooks:** `analytics.risk_observations`, `analytics.risk_model_evaluations`, dispersion–AQ comparison tables with **evidence labels**, SQL views, optional **aligned fixtures**, and **dry-run-friendly** producers.
- **Alerting scaffolding** (`analytics.fn_alert_candidates`, materialized incidents, notifiers) with **runbook mappings** under `docs/runbooks/`.
- **Phase 13 tooling:** GitHub Actions CI, optional Compose integration workflow, **minimal census bootstrap** for bounded tests, **immutable calibration CSV/optional Parquet exports**, **`make release-check`**, and **`make version`**.

## What this release does **not** claim

- **No scientific validation** of smoke exposure, health outcomes, or forecast skill.
- **No regulatory air-quality modeling** — dispersion and lag summaries are **triaging aids**, not proof of causality or compliance-grade performance.
- **No guarantee** that sparse dashboards or “green” checks imply the risk model is correct; **no data** and **insufficient sample** states must be read literally.

## Data sources

- **NASA FIRMS** (live requires `FIRMS_MAP_KEY`; fixtures via dry-run).
- **OpenAQ v3** (optional API key depending on access; fixtures via dry-run).
- **NWS** wind / gridpoint pathways when explicitly configured and bounded.
- **U.S. Census TIGER/Line** boundaries for operational bootstrap (`make db-bootstrap`) — **not** used when running **`make db-bootstrap-minimal`** (synthetic GeoJSON only).

## Operational modes

- **Fixture / dry-run:** default demos and CI-friendly paths avoid secrets and external APIs.
- **Live ingest:** bounded bboxes and explicit env guards; operators supply keys.
- **Compose stack:** Postgres + Redpanda (+ optional Grafana / scheduler profiles).
- **Exports:** `make export-calibration*` writes **immutable review snapshots** under `artifacts/calibration/<UTC-stamp>/` (CSV required; Parquet if `pyarrow` is installed).

## Known limitations

- Correlation and precision/recall-like metrics are **suppressed** when match counts are below thresholds — low **`match_count`** means **do not** interpret Pearson **`correlation`** as meaningful.
- **Minimal census geometries** cover only a handful of synthetic counties/tracts for CI/integration — they are **not** interchangeable with real TIGER boundaries for production mapping.
- **Integration workflow** is **heavy** (Compose + Spark profile); keep it **manual / scheduled / label-gated**, not on every PR push.

## Validation commands

```bash
uv sync --extra dev
uv run ruff check src tests
uv run pytest -q
python3 -m json.tool docker/grafana/dashboards/smoke-risk.json >/dev/null
bash -n scripts/*.sh
SMOKE_NO_COMPOSE=1 bash scripts/smoke_test.sh
```

With Compose up, migrations applied, and fixtures aligned:

```bash
make smoke-test
make integration-smoke-test
COMPOSE_INTEGRATION=1 make release-check   # includes integration-smoke-test
make version
make export-calibration
```

## Quickstart (operators)

See **`README.md`** — typical path: `make up` → `make topics` → `make db-bootstrap` (or **`make db-bootstrap-minimal`** for bounded tests only) → replay fixtures / demos.

## No-secrets demo path

Use **`FIRMS_DRY_RUN=1`**, **`OPENAQ_DRY_RUN=1`**, **`WIND_DRY_RUN=1`**, **`GRID_WEATHER_DRY_RUN=1`** as documented in **`.env.example`**, plus **`make demo`** / **`make replay-fixtures`** / **`make calibration-demo`** (last is Spark-heavy).
